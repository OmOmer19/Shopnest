## 📖 About

**Shopnest** is a sleek, fully-featured e-commerce web application built on the **MERN stack**. It delivers a seamless shopping experience — from browsing products and managing your cart, all the way to secure checkout with payment integration.

Whether you're a customer exploring products or a developer looking to understand full-stack e-commerce architecture, Shopnest has you covered.

> 🔗 **Live at:** [shopnest-eta.vercel.app](https://shopnest-eta.vercel.app)

---

## ✨ Features

| Feature | Description |
|---|---|
| 🔐 **User Auth** | Secure signup & login with JWT-based authentication |
| 🛒 **Cart & Checkout** | Add to cart, update quantities, and proceed to checkout |
| 💳 **Payment Gateway** | Integrated payment processing for real transactions |
| 📱 **Fully Responsive** | Optimized for mobile, tablet, and desktop |
| ⚡ **Blazing Fast** | Powered by Vite for lightning-fast dev & production builds |
| 🎨 **Modern UI** | Clean, minimal design built with Tailwind CSS |

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:
- [Node.js](https://nodejs.org/) (v18 or higher)
- [MongoDB](https://www.mongodb.com/) (local or Atlas)
- [Git](https://git-scm.com/)

### 1. Clone the Repository

```bash
git clone https://github.com/OmOmer19/Shopnest.git
cd Shopnest
```

### 2. Setup the Backend

```bash
cd Backend
npm install
```

Create a `.env` file in the `Backend` folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret_key
```

Start the backend server:

```bash
npm run dev
```

### 3. Setup the Frontend

```bash
cd ../Frontend
npm install
```

Create a `.env` file in the `Frontend` folder:

```env
VITE_API_URL=http://localhost:5000
```

Start the frontend:

```bash
npm run dev
```

### 4. Open in Browser

```
http://localhost:5173
```
